# Product Overview

This is an AWS Machine Learning Bootcamp repository tracking a 12-week journey learning AI & Machine Learning on AWS. The project focuses on building foundational ML knowledge through hands-on implementation of algorithms from scratch, progressing to AWS-specific ML services and tools.

## Key Goals
- Implement core ML algorithms (linear regression, logistic regression, etc.) from scratch using NumPy
- Learn AWS ML services including SageMaker, Glue, Comprehend, Rekognition, and Bedrock
- Build 5+ portfolio projects demonstrating ML concepts
- Prepare for AWS Certified ML Specialty Exam

## Learning Path
The bootcamp follows a structured 12-week curriculum covering ML foundations, data preparation, deep learning, NLP, generative AI, and MLOps practices.

## Target Audience
Students and practitioners learning machine learning with a focus on AWS cloud services and certification preparation.
